常用命令参考

快速查找常用命令，点击分类查看详细说明

openclaw setup
入门

初始化配置和工作区，首次安装必运行

openclaw setup
openclaw onboard
入门

交互式向导，快速设置 Gateway、工作区和技能

openclaw onboard
openclaw status
入门

查看会话健康状况和最近联系人

openclaw status
openclaw doctor
入门

健康检查和快速修复，诊断常见问题

openclaw doctor
gateway start
中级

启动 Gateway 服务

openclaw gateway start
gateway stop
中级

停止 Gateway 服务

openclaw gateway stop
gateway restart
中级

重启 Gateway 服务

openclaw gateway restart
gateway status
中级

查看 Gateway 运行状态

openclaw gateway status
logs
中级

查看 Gateway 日志（实时跟踪）

openclaw logs --follow
channels add
中级

添加新的聊天通道

openclaw channels add --channel telegram --token $TELEGRAM_BOT_TOKEN
channels list
中级

列出所有已配置的通道

openclaw channels list
channels status
中级

检查通道连接状态和健康度

openclaw channels status --probe
channels remove
高级

移除已配置的通道（使用 --delete 彻底删除）

openclaw channels remove --channel discord --delete
models status
中级

查看当前配置的模型和认证状态

openclaw models status
models list
中级

列出所有可用模型

openclaw models list --all
models set
中级

设置默认模型

openclaw models set anthropic/claude-3.5-sonnet
models auth
高级

添加或管理模型认证（API Key、Token）

openclaw models auth add --provider anthropic
memory search
中级

在 MEMORY.md 和 memory/*.md 中进行语义搜索

openclaw memory search "项目进展"
memory status
中级

查看记忆索引统计信息

openclaw memory status
memory index
高级

重新索引记忆文件

openclaw memory index
cron add
高级

添加定时任务（提醒、周期性任务）

openclaw cron add --name "晨会提醒" --at 09:00 --message "开会啦"
cron list
高级

列出所有定时任务

openclaw cron list --all
cron rm
高级

删除指定的定时任务

openclaw cron rm
cron runs
高级

查看定时任务执行历史

openclaw cron runs --id --limit 10
